# Example Files

This directory contains example files for the Game Config Text Checker.

## Files

- `sample_data.xlsx` - Sample Excel file for testing (create your own)
- `sample_report.xlsx` - Sample output report (generated by the tool)

## Creating Sample Data

To create a sample data file:

1. Create an Excel file with the following structure:

| id | text |
|----|------|
| 1 | 你好，欢迎来的洛克王国！ |
| 2 | 这是一段测试文本。 |
| 3 | 快递的包裹送到了。 |

2. Save as `sample_data.xlsx`

3. Run the checker:

```bash
python conf_check.py examples/sample_data.xlsx Sheet1 text
```

## Expected Output

The checker will generate a report identifying issues like:
- Line 1: 错别字 - "来的" should be "来到"
- Line 3: 错别字 - "的" should be "地" (快递地包裹)
